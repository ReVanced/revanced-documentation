# Building the ReVanced Patcher

1. Clone the repository

   ```bash
   git clone https://github.com/revanced/revanced-patcher && cd revanced-patcher
   ```

2. Publish the patcher to your local maven repository

   ```bash
   ./gradlew publish
   ```
