# Building the ReVanced Patches

1. Clone the repository

   ```bash
   git clone https://github.com/revanced/revanced-patches && cd revanced-patches
   ```

2. Build the patches

   ```bash
   ./gradlew build
   ```

The file will be located in `build/libs/revanced-patches-VERSION.jar`
